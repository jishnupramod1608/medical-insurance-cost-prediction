# Medical Insurance Cost Prediction

## Objective
To predict medical insurance charges based on demographic and lifestyle factors using machine learning.

## Dataset
Medical Cost Personal Dataset (Kaggle)

## Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Google Colab

## Models Used
- Linear Regression
- Random Forest Regressor

## Conclusion
Smoking status, age, and BMI are the most influential factors affecting insurance costs.
Random Forest performed better than Linear Regression.
